基本概念

  • Workspace：工作区，能看到的目录
  • Index / Stage：暂存区
  • Repository：仓库区（或本地仓库）
  • Remote：远程仓库

HEAD: 指向该分支上的最后一次提交，待补充

master: 只是这个repository中默认创建的第一个branch

origin: 远程仓库在本地的标签或者别名，git clone url 默认创建的指向这个远程代码库的标签
