函数

$(function <arg>) 或 ${function <arg>}

字符串操作

函数 名称 功能 返回
$(subst <from>,<to>,<text>) 字符串替换 把text里from替换成to 替换后的字符串
$(patsubst <pattern>,<replacement>,<text>) 模式字符串替换 text中以空格、回车、TAB是否符合pattern, 若符合, 用replacement替换 替换后的字符串
$(strip <string>) 去空格 去掉开头和结尾空字符 处理后的字符串
$(findstring <find>,<in>) 查找字符串 在in中查找find 找到返回find，否则返回空
$(filter <pattern…>,<text>) 过滤函数 保留符合pattern的单词，可以有多个模式 符合的字符串
$(filter-out <pattern…>,<text>) 去除函数 去除符合模式的单词 处理后的字符串
$(srot <list>) 排序 单词按升序排 排序后字符串
$(word <n>,<text>) 取单词 取第n个单词, 从1开始 单词或空
$(wordlist <n>,<m>,<text>) 取单词 去n到m的单词 单词或空
$(words <text>) 统计个数 统计单词个数 单词数量
$(firstword <text>) 取首个单词 等价于$(word 1,<text>) 首个单词

文件名操作

函数 名称 功能 返回
$(dir <names…>) 取目录 取出names反斜杠的目录 目录或./
$(notdir <names…>) 取文件名 去掉反斜杠前面的目录部分 文件名序列
$(suffix <names…>) 取文件序列后缀 去掉非后缀部分 .cpp等后缀 或 空
$(basename <names…>) 取文件序列前缀 去掉后缀部分
$(addsuffix <suffix>,<names…>) 加后缀 加后缀 文件序列名
$(addprefix <prefix>,<names…>) 加前缀 加前缀 文件序列名
$(join <list1>,<list2>) 链接 list1+list2 链接后的字符串

控制和调用

  • $(foreach <var>,<list>,<text>)
    循环执行

  • $(if <condition>,<then-part>)

  • $(call <expression>,<param1>,<param2>…)
    创建新的函数

  • $(origin <var>)
    变量是哪里来的, undefined, default, file, command line, override, automatic

shell函数

text := $(shell cat foo)

控制Make函数

函数 名称 功能 返回
$(error <text…>) 产生一个致命错误 提示错误信息 提示
$(warning <text…>) 输出告警信息 提示告警信息 提示
